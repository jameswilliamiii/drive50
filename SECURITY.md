# Security Policy

## Reporting a Vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report them privately through GitHub's [private vulnerability reporting](https://github.com/jameswilliamiii/drive50/security/advisories/new).
This opens a confidential advisory visible only to you and the maintainers.

We aim to acknowledge reports within a few days and will keep you updated as we
investigate and prepare a fix. Once a fix is released, we're happy to credit you
in the advisory unless you'd prefer to remain anonymous.

## Scope

This is a personal project. There is no bug-bounty program and no guaranteed
response time, but security reports are genuinely appreciated and taken seriously.
